# Gradient descent of knot energy functional
* 本プロジェクトは Unity バージョン 2019.2.21f1 で作成されました．
* 結び目のエネルギー関数の勾配流に沿って結び目を変形させる Unity プログラムです．

# How to Use
* スペースキーを押している間だけ結び目が更新されます．
